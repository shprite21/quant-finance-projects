# Quantitative Finance Modeling

A collection of core quantitative finance projects covering probabilistic modeling, portfolio optimization, and time-series forecasting.

These projects focus on both implementation and critical evaluation of models used in financial markets.

---

## Overview

This repository includes:

* **Monte Carlo Risk Simulation** — modeling uncertainty in asset returns and estimating tail risk
* **Portfolio Optimization** — constructing efficient portfolios using mean–variance theory
* **ARIMA Forecasting** — building and evaluating statistical time-series models

Each project emphasizes:

* Mathematical formulation
* Python implementation
* Model validation
* Limitations and real-world considerations

---

## Projects

### 1. Monte Carlo Risk Simulation

Simulates thousands of future price paths using estimated return distributions to quantify risk via Value at Risk (VaR) and Expected Shortfall (ES).

→ Highlights limitations of normality assumptions and explores tail risk.

---

### 2. Mean-Variance Portfolio Optimization

Constructs efficient portfolios using covariance estimation and constrained optimization.

→ Demonstrates diversification, efficient frontier, and instability of optimal weights.

---

### 3. ARIMA Time-Series Forecasting

Builds ARIMA models to forecast asset prices and evaluates performance against naive baselines.

→ Focuses on model validation and predictive limitations.

---

## Project Philosophy

Each project is implemented in two stages:

1. Baseline model — core theoretical implementation  
2. Enhanced model — improvements addressing real-world limitations  

This approach reflects how quantitative models are developed and refined in practice.
